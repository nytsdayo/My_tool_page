mlx_init

Setup

概要

void *mlx_init(void);

説明

MiniLibXの初期化を行います。描画を開始するための内部データとコンテキストを作成します。

戻り値

成功時にMiniLibXの識別子（ポインタ）を返します。失敗時はNULL。

mlx_new_window

Setup

概要

void *mlx_new_window(void *mlx_ptr, int size_x, int size_y, char *title);

引数

mlx_ptr
mlx_initで取得した識別子。
size_x
ウィンドウの幅。
size_y
ウィンドウの高さ。
title
ウィンドウのタイトルバーに表示される文字列。

戻り値

作成したウィンドウの識別子を返します。

mlx_loop

Loop

概要

int mlx_loop(void *mlx_ptr);

説明

イベントループを開始します。この関数は無限ループに入り、イベントが発生するたびに登録されたフック関数を呼び出します。

mlx_loop_end

Loop

概要

int mlx_loop_end(void *mlx_ptr);

説明

イベントループを終了させます。

mlx_loop_hook

Hooks

概要

int mlx_loop_hook(void *mlx_ptr, int (*funct_ptr)(), void *param);

説明

イベントが発生していない時に繰り返し呼び出される関数を登録します。

mlx_hook

Hooks

概要

int mlx_hook(void *win_ptr, int x_event, int x_mask, int (*funct)(), void *param);

引数

x_event
X11イベントコード（例: KeyPress=2, DestroyNotify=17）。
x_mask
イベントマスク（通常は0または該当マスク）。
funct
呼び出されるコールバック関数。

mlx_get_data_addr

Images

概要

char *mlx_get_data_addr(void *img_ptr, int *bits_per_pixel, int *size_line, int *endian);

説明

画像のピクセルデータへのポインタを取得します。直接ピクセルを操作する場合に使用します。

引数

bits_per_pixel
1ピクセルあたりのビット数（通常32）。
size_line
画像の1行あたりのバイト数。
endian
ピクセルデータのエンディアン。
