# voice_input_page

音声入力ができるシンプルなWebページです。

## GitHub Pages

このサイトはGitHub Pagesでホストされています。

### セットアップ

1. リポジトリの Settings > Pages に移動
2. Source で "GitHub Actions" を選択
3. main ブランチにプッシュすると自動的にデプロイされます

### 使い方

1. マイクボタンをクリックして音声入力を開始
2. 日本語で話すと、音声が文字に変換されます
3. もう一度ボタンをクリックすると停止します

### 対応ブラウザ

- Google Chrome (推奨)
- Microsoft Edge
- Safari (iOS)
